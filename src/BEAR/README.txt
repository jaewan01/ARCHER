BEAR is a method for fast, scalable, and accurate random-walk-with-restart computation on large graphs.
The details of BEAR can be found in the following paper:

    BEAR: Block Elimination Approach for Random Walk with Restart on Large Graphs

1. How to compile :

	run 'Mex_compile.m' in Matlab

2. How to run an example code :
	
	run 'Example.m' in Matlab
